# ArchHypo.AI Plugin

ArchHypo.AI is a plugin designed to assist software architecture decision-making using Large Language Models (LLMs). It integrates directly with Trello to help teams manage architectural hypotheses, technical plans, and uncertainty metrics in real-time.
